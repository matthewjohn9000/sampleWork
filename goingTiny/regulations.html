Parking Your Tiny House Legally

So you want to build a tiny house and you'd like to live in it legally. Gathering information is a key to success and is the reason you are visiting this website, right?To begin, there are two very important aspects to legally parking and living in a tiny house:

  1. Build your dwelling so that it is compliant with the local building code where it will be sited. For example, a house on a foundation has different rules than a house on a trailer with wheels.
  2. Build your house so that it meets the local zoning requirements. Will it be an ADU/DADU (Accessory Dwelling Unit) in someone's backyard or are you hoping to install it in a park or on a plot of land with other tiny houses?

Building Your Tiny House to Code

The take away is that the house needs to be built to code and sited legally in the jurisdiction. For the house to be built to code depends on which one of the following three categories it could fall under:

  1. Attached to a foundation.
  2. Resting on a trailer, mobile like a recreational vehicle.
  3. Modular, built off-site but then moved to the residents location.

All three of these categories have varying size or square footage requirements to be legal, and their rules vary: For example, if you build a tiny house under the local modular housing guidelines, you have to use R21 insulation in the walls.

R21 is 5-1/2” thick which means it needs to be placed in a wall built with 2” X 6” studs. This is in contrast to some manufactured homes which are only required to have 2” X 4” studs in their exterior walls. These are the type of details you will need to learn and document to meet code approval.
